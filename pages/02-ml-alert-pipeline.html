步骤 02

机器学习告警管线

AutoEncoder 和 Graph Model 先产出风险信号，再融合成标准告警。

检测层输出 模型产物
01
AutoEncoder 管线 交易、画像、设备 → 异常分数。
02
Graph Model 管线 关系图 → 共享设备、资金链路、高风险邻居。
03
融合生成告警 合并分数，输出 risk_score、risk_type 和标准告警。
04
前 N 个触发特征 金额、次数、历史阈值三类触发原因。
标准化告警对象 下一步输入
alert_id 关联证据、查询、报告和审计轨迹。
account_id 与 alert_time 限定取证对象和时间窗。
risk_score 与 risk_type 决定调查优先级和证据路径。
top_features 每个触发特征后续都要核验。
边界清楚 机器学习模型发现异常，不写报告。
为什么重要 Agent 接住可解释触发特征。
Agent Loop 接入点 编排器决定查什么、调什么、何时停。
