步骤 11

MLOps 与生产可靠性

生产关注四件事：监控、降级、回滚、审计完整性。

生产可靠性 监控 + 降级 + 回滚
01
数据质量监控 延迟、缺失率、schema、特征新鲜度、data/feature drift。
02
模型健康监控 score 分布、risk_type、阈值命中率、标签回流。
03
每周模型重训 延迟标签 + 人审结果 + 数据快照，过评估再发布。
04
Agent 运行监控 Tool 成功率、SQL 超时、queue、p95、cost、退回率。
05
审计日志监控 Trace、query_hash、Model/Prompt/Evidence version 完整。
降级路径 异常时回退到规则、人工队列、稳定模型或只读查询。
版本回滚 记录模型、Prompt、工具、数据快照，灰度失败即回滚。
数据库灾备 交易库、Case Store、对象存储备份与恢复演练。
上线策略 影子运行 → 小流量灰度 → 扩大覆盖。
