步骤 03

LangGraph-style Agent Loop

一个告警进入 StateGraph；证据不足继续取证，执行失败进入重试，证据齐全进入报告复核。

StateGraph 工作流 节点 + 条件路由
START
边
函数节点 Context 构建器 告警、特征、Tool schema。
State
Agent 节点 Evidence Router Agent 选择证据路径并输出 tool_call。
tool_call
函数节点 Guardrail 校验门 校验 schema、权限、时间窗、图深度、预算。
函数节点 Tool 执行器 Database 查询 / Graph 搜索 / Policy 检索 / Python 计算。
Observation
函数节点 Evidence 写入器 工具结果 → Evidence card。
State
路由节点 Route(state) 按缺证、拦截、证据齐全分流。
Route(state) 条件边 每一条边都决定下一跳
↺ Evidence Router Agent 证据缺口存在
↺ 修复 → Router Guardrail 未通过 / Tool 错误
→ Report Agent → 人工复核 证据齐全
