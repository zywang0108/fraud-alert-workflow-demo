Guardrails / 权限边界

所有 tool_call 先过受限数据库、工具白名单、数据范围、成本和人工边界校验。

Tool 调用校验门 执行前校验
Agent 输出 tool_call 请求 tool_name、arguments、case_id、account_id、window、reason。

Guardrail 校验 执行前验证
受限 Database只允许访问受限视图、字段和 query_template。
Tool 白名单只允许调用当前案件注入的 tool_name。
权限校验 role、tenant、case、risk_type。
数据范围限制账户、时间窗、row_limit、depth。
执行预算限制 timeout、retry 和 token cost。
人工边界冻结、关闭、申报必须人工确认。
→
运行框架动作 允许或拦截
通过：执行工具，写 Observation、Trace、Evidence。
